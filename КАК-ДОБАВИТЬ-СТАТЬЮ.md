# Как добавить статью в блог

## Шаг 1. Создай файл

В папке `posts/` создай файл с именем на латинице через дефисы — оно станет адресом страницы.
Например: `posts/why-ai-audits-win.md` → статья будет на `/blog/why-ai-audits-win.html`.

## Шаг 2. Заполни шапку и текст

Файл начинается с шапки между `---`, дальше идёт сам текст:

```
---
title: Заголовок статьи
date: 2026-07-15
description: Одно-два предложения — они показываются в списке блога и в Google.
---

Первый абзац статьи.

## Подзаголовок раздела

Обычный текст. **Жирный** и *курсив*. Ссылка: [Savant.chat](https://savant.chat).

- пункт списка
- ещё пункт

> Цитата — выделяется оранжевой рамкой.
```

Правила шапки:
- `date` строго в формате ГГГГ-ММ-ДД — по ней сортируется список (новые сверху).
- Если в `title` или `description` есть двоеточие — это нормально, всё сработает.
- Необязательное поле `meta_extra:` — приписка после даты (например, «based on my talk …»).

## Шаг 3. Запусти сборку

Дважды кликни файл **«Обновить блог.command»** в папке сайта
(или в Терминале: `python3 build.py`).

Скрипт сам создаст страницу статьи в дизайне сайта, обновит список в блоге и sitemap.xml.

## Шаг 4. Задеплой

Пока сайт не задеплоен — просто попроси Клода: «задеплой сайт».
После настройки хостинга здесь появится команда/кнопка для публикации.
